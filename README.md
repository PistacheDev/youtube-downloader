# Youtube Downloader
Télécharger gratuitement vos vidéos YouTube !

## Important
Ce projet est Open Source est vous pouvez donc le mettre à jour et faire un pullrequest !

## Requis
- Python (3.10 ou plus)
- ffmpeg

## Installation de ffmpeg
- Linux :
```sh
sudo apt-get install ffmpeg
```
- Windows : https://www.youtube.com/watch?v=DMEP82yrs5g

## Installation
1) Téléchargez le downloader.
2) Et c'est tout ! Vous avez juste à profiter !

## Lancement
Exécuter le programme qui porte le nom de votre OS (Windows ou Linux).
Si vous avez Python (portable par exemple), mais qu'il ne vous donne pas la commande "python" dans votre environnement :
1) Ouvrez le logiciel interprète.
2) Entrez y la commande :
```sh
python chemin/vers/index.py
```
