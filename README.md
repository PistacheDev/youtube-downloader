Ce programme permet de télécharger gratuitement une vidéo YouTube avec un programme en Python.

Modules :
- "pytube"
- "moviepy"
- Dépendances des modules cités prédemment.

Prérequis :
- Vous devez avoir au préalable, une version récente de Python (portable ou non) installée sur votre ordinateur.
- La vidéo que vous voulez télécharger ne doit pas être bloquée par votre administrateur réseau.

- Fonctionnement :
Si le fonctionnement vous intéresse, le code télécharges la vidéo et l'audio séparemment avec "pytube" (car certaines résolutions n'ont pas d'audio en téléchargement) et les assemble avec "moviepy".

Merci d'avoir installé ce programme pour télécharger vos vidéos YouTube !
