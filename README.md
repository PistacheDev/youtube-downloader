Salut, ce programme a été développé par Pistache !

Il permet de télécharger gratuitement, avec un programme en Python et le module Python "pytube", une vidéo YouTube.
Vous devez avoir au préalable, une version récente de Python (portable ou non) installée sur votre ordinateur.

IMPORTANT :
Si vous ne renseignez aucun chemin d'accs personnalisé lors du téléchargement de la vidéo, il sera enregistré dans le dossier "downloads".

Merci d'avoir installé ce programme pour télécharger vos vidéos YouTube !
