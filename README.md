Salut !
Ce programme a été développé par Pistache !

Il permet de télécharger gratuitement avec un programme en Python, une vidéo YouTube, en fonction de son lien.
Vous devez avoir au préalable, une version récente de Python d'installé sur votre ordinateur et d'avoir installé le paquet Python "pytube".

Le batch (PyYT.bat) est réservé pour Windows.
Le Script Shell (PyYT.sh) est réservé pour Linux (VPS par exemple).

Merci d'avoir installé ce programme pour télécharger vos vidéos YouTube !
